# koa-rest-crud-jwt

Test application based on REST API with creating user, login and posts.

## Installation

```bash
npm install  || npm i
```
##### All local ENV are in nodemon.json
##### Running in dev mode with nodemon
```
npm run dev
```

##### ApiDoc generation
```bash
npm run apidoc
```
